# Security Policy

## Supported Versions

Currently, only the `main` branch (latest release) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for MWareX. If you discover a vulnerability, we would like to know about it so we can take steps to address it as quickly as possible.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to us via email at **security@mwarex.in**. 

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible and acknowledge your contribution in our release notes (if desired).
